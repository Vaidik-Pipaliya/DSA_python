## 📖 Overview

Welcome to my **DSA Python** learning repository! This project serves as a personal knowledge base and practical playground for mastering **Data Structures**, **Algorithms**, and essential Python programming techniques. It includes various Jupyter Notebooks covering everything from basic algorithms to practical, real-world mini-projects.

## 📂 Repository Structure

### 🔍 Algorithms
* **Searching Algorithms:**
  * [`linear.ipynb`](linear.ipynb) - Implementation of Linear Search.
  * [`binSearch.ipynb`](binSearch.ipynb) - Implementation of Binary Search.
* **Sorting Algorithms:**
  * [`bubblesort.ipynb`](bubblesort.ipynb) - Bubble Sort algorithm.
  * [`selectionSort.ipynb`](selectionSort.ipynb) - Selection Sort algorithm.

### 🏗 Data Structures
* [`array.ipynb`](array.ipynb) - Exploring array manipulations and operations.

### 🧠 Core Python Concepts
* [`function.ipynb`](function.ipynb) & `function_examples/` - Deep dive into Python functions.
* [`lambda.ipynb`](lambda.ipynb) - Anonymous functions in Python.
* [`map.ipynb`](map.ipynb) - Using the `map()` function for iterables.
* [`filter.ipynb`](filter.ipynb) - Filtering data effectively using `filter()`.

### 🛠 Mini Projects
Applied learning through practical notebook-based projects:
* [`inventory.ipynb`](inventory.ipynb) - A simple inventory management system.
* [`organizning_student_grades.ipynb`](organizning_student_grades.ipynb) - Script to calculate and organize student grades.
* [`to_do_list.ipynb`](to_do_list.ipynb) - A functional to-do list application.
* [`userfeedback.ipynb`](userfeedback.ipynb) - Collecting and processing user feedback data.

## 🚀 How to Use

1. **Clone the repository:**
   ```bash
   git clone https://github.com/your-username/DSA_python.git
   ```
2. **Navigate to the directory:**
   ```bash
   cd DSA_python
   ```
3. **Open Jupyter Notebook:**
   ```bash
   jupyter notebook
   ```
4. **Explore the files!** Open any `.ipynb` file to run the cells, learn from the code, and experiment on your own.
